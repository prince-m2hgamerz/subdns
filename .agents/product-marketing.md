# SubDNS — Product Marketing Context

## Product Overview
SubDNS is a free subdomain management platform. Claim subdomains under `m2hio.in` with full DNS control, Cloudflare proxying, automatic SSL — no credit card required.

**URL:** https://subdns.m2hio.in  
**Tagline:** Your corner of the internet. Free forever.  
**Status:** Launched. Active development.

---

## Core Positioning

**One-liner:** *Free subdomains on m2hio.in with full DNS control, Cloudflare global edge, CLI & API access.*

**Elevator Pitch:** Every developer deserves a corner of the internet to call their own. SubDNS gives you free subdomains under m2hio.in — no credit card, no domain purchase, no Cloudflare access needed. Claim a subdomain, add it to your hosting provider (Vercel, Netlify, Railway, etc.), and you're live.

**Key Differentiators:**
- **Free forever** — no credit card, no time limit, no tricks
- **Developer-first** — CLI tool and REST API for programmatic control
- **Zero setup** — Cloudflare edge, automatic SSL, DDoS protection included
- **Multiple tiers** — Free → Pro ($9/mo) → Team ($29/mo) → Enterprise (custom)

---

## Ideal Customer Profile (ICP)

### Primary: Solo Developers & Indie Hackers
- Building side projects, MVPs, personal sites, or portfolio pages
- Don't want to pay for a domain until they know the project has traction
- Comfortable with CLI/API
- Deploy to Vercel, Netlify, Railway, GitHub Pages, Fly.io, etc.

### Secondary: Small Teams & Startups
- Need subdomains for staging/preview environments (e.g., `staging.project.m2hio.in`)
- Multiple devs needing quick, throwaway subdomains
- Need activity logs, webhooks, and collaboration

### Tertiary: Enterprise / Large Orgs
- Need custom domain (white-label), SSO, dedicated infrastructure
- Compliance requirements (SOC 2, GDPR)
- Team workspaces with granular permissions

## Competitive Landscape

**Direct Competitors:**
- **freedns.afraid.org** — Free DNS, older UI, no Cloudflare
- **nip.io / sslip.io** — Wildcard DNS, no management UI
- **xip.io** — Similar to nip.io, dead project
- **Cloudflare DNS** — Powerful but requires owning a domain
- **Namecheap FreeDNS** — Limited, ad-supported

**Indirect:**
- **Vercel / Netlify preview domains** — Temporary, platform-locked
- **ngrok** — Tunneling, different use case
- **Local development tools** — Don't solve the "public URL" problem

**SubDNS Advantage:** Free, instant, Cloudflare-powered, developer tools (CLI + API), no lock-in, no domain ownership required.
